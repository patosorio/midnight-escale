## Getting Started

### Prerequisites

- Node.js (v18 or higher)
- npm or yarn

### Installation

1. Clone the repository:
```bash
git clone [your-repository-url]
cd midnightescales
```

2. Install dependencies:
```bash
cd client
npm install
```

3. Run the development server:
```bash
npm run dev
```

The application will be available at `http://localhost:3000`.

## Development

The project uses:
